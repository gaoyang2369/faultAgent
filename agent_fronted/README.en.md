# Agent Frontend

This is a frontend project based on Vue 3, built with Vite and developed using TypeScript, primarily designed to implement an intelligent customer service system.

## Features

- Intelligent chat interface
- Conversation history management
- Customizable message styling
- Code syntax highlighting
- Dark mode support
- PDF file viewing

## Technology Stack

- Vue 3 (Composition API)
- TypeScript
- Vite
- Pinia (State Management)
- Vue Router
- Sass/CSS

## Directory Structure

```
src/
├── assets/               # Static assets
├── components/           # Vue components
├── hooks/                # Custom hooks
├── router/               # Routing configuration
├── services/             # API services
├── stores/               # Pinia state management
├── type/                 # TypeScript type definitions
├── utils/                # Utility functions
├── views/                # Page views
├── App.vue               # Root component
├── main.ts               # Entry file
├── vite.config.ts        # Vite configuration
├── tsconfig.json         # TypeScript configuration
```

## Installation and Setup

1. Clone the repository
2. Run `npm install`
3. Start the development server with `npm run dev`

## Usage Instructions

This project primarily implements an intelligent customer service interface featuring a conversation window, message bubbles, and code block display. It is developed using Vue 3's Composition API, manages state with Pinia, and handles navigation with Vue Router.
