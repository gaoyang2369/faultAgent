# FileUpload 上传界面整理计划

## 目标

整理当前 `agent_fronted/src/views/FileUpload.vue` 上传界面体验，保持默认聚焦上传和识别结果，同时保留历史记录、PDF 原件预览和导出能力。

## 范围

- 历史记录默认隐藏，通过左上角边缘按钮展开。
- 历史记录隐藏时，上传区和识别结果区为 5:5 比例。
- 历史记录展开时保持三栏布局效果。
- 上传 PDF 后在上传面板内展示原始 PDF 内嵌预览。
- 保持上传、登记、结果预览、历史记录选择与清空逻辑不变。

## 验证

- 前端构建通过。
- 历史记录按钮可切换展开/隐藏状态，视觉入口不干扰主体内容。
- 桌面端隐藏历史记录后左右两栏等宽，展开后恢复三栏布局。
- PDF 原件预览、登记、结果预览和打印导出可用。
