# 咨询记录管理交互总结

## 完成内容

- 左侧咨询记录列表新增更多操作菜单。
- 支持本地即时重命名，并通过 `localStorage` 保持刷新后名称不丢失。
- 支持真实删除：
  - 本地缓存记录直接删除浏览器持久化内容
  - 服务端 thread 记录调用最小后端删除接口
- 删除当前激活会话时会安全切换到下一条记录或回到空会话态。
- 移除了右上角重复的独立声纹认证按钮，保留管理员身份入口打开声纹认证弹窗。

## 额外修复

- 兼容旧浏览器缓存里残留的历史 `thread.*`：
  - 当前会话无权删除的旧 thread 不再一律弹失败
  - 会先清理本地残留缓存，避免记录“删不掉又一直出现”

## 验证

- 前端构建通过。
- 后端删除接口可用。
- 本地重命名 / 删除在刷新后保持正确。
- 旧 thread 残留缓存删除逻辑已复测。
- 管理员入口仍可正常进入声纹认证窗口。
