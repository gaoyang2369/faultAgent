# 语音提问重复回复排查计划

- 保持文字提问链路不变。
- 优先用最小前端修复避免重复渲染和重复播放。

- 前端构建通过。
- 静态检查关键事件分支，确认 `asr_result` 后语音网关 Agent/TTS 事件不会再生成第二条助手消息。
