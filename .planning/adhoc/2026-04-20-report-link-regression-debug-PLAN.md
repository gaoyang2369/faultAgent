# 2026-04-20 报告链接点击回归排查计划

## 背景

- 用户反馈聊天回答末尾的“【报告文件】…html”不再可点击。
- 该能力过去可用，属于已有链路回归，不是新功能缺失。

## 排查范围

1. 报告文件是否正常生成并落在 `agent_fronted/public/reports/`
2. 后端 `/reports/*` 静态访问是否正常
3. Agent 最终消息中的报告文本格式是否发生变化
4. 前端 `ChatMessage.vue` 是否仍能识别并渲染报告链接
5. 实时消息与历史恢复是否共用同一套报告链接识别逻辑

## 目标

- 用最小侵入修复“报告文件文本 -> 可点击链接 -> 安全打开报告页面”的闭环
- 不开放任意 HTML / 任意路径访问
- 保持 SSE、history、报告生成主链路不回退
