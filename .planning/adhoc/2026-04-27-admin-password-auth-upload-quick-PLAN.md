# 管理员密码登录与服务端 PDF 上传链路补齐计划

## 目标

在声纹认证后端尚未可用的情况下，补一条最小可联调的管理员认证与服务端 PDF 上传链路，确保上传记录、原文件访问和删除都走后端真实权限控制。

## 范围

- 新增基于用户名/密码的最小管理员登录接口。
- 复用现有会话体系，签发与 `fd_session` 绑定的管理员 cookie。
- 将 PDF 上传历史从前端本地缓存切到后端真实登记。
- 提供管理员上传记录的查询、原文件访问和删除接口。
- 上传文件与元数据落到受控运行目录，避免污染仓库业务目录。

## 风险控制

- 不在前端保存管理员密码。
- 不开放无鉴权的上传 / 删除能力。
- 声纹认证窗口和后续正式权限模型暂不在本轮展开，只保留可复用入口。

## 验证

- 未登录访问管理员上传接口返回 `403`。
- 密码登录成功后刷新页面仍保持管理员态。
- PDF 上传、历史加载、原文件访问、删除链路可真实工作。
- 普通聊天与 SSE 主链路不回退。
