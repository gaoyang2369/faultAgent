Create `tests/test_history_api.py` implementing SAFE-03 history API tests.

**Critical detail:** The `/ai/history/{type}` endpoint uses `async for checkpoint_tuple in request.app.state.checkpointer.alist()`. This means `checkpointer.alist()` must return an **async iterable**. A plain `MagicMock` won't work with `async for`. We need to configure the mock to return an async generator.

**Mock setup for each test (or in a local fixture):**
```python
async def mock_alist_generator(*args, **kwargs):
    """模拟 checkpointer.alist() 返回 async generator。"""
    mock_ct_1 = MagicMock()
    mock_ct_1.config = {"configurable": {"thread_id": "thread-1"}}
    mock_ct_2 = MagicMock()
    mock_ct_2.config = {"configurable": {"thread_id": "thread-2"}}
    yield mock_ct_1
    yield mock_ct_2
```

Assign: `test_client.app.state.checkpointer.alist = mock_alist_generator`

For `checkpointer.aget`, it's a regular async function, so `AsyncMock(return_value=...)` works.
