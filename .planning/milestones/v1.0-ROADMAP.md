# Roadmap: AI Agent 后端模块化重组

## Overview

将 592 行单体 app.py 和 597 行 tools.py 重组为清晰的模块结构。通过四个 Phase 逐步拆分，每个 Phase 结束时系统处于可运行状态。Phase 1 已完成特征测试安全网，Phase 2-5 依次提取 config/utils、tools、prompts/middleware/KB、最终瘦身 app.py。

## Phases

- [x] **Phase 1: Safety Net** - 建立特征测试套件和安全前置条件
- [x] **Phase 2: Config & Utils** - 提取配置和通用工具函数
- [x] **Phase 3: Tools Modularization** - 拆分 tools/ 目录 + 延迟初始化 + 子agent迁移
- [ ] **Phase 4: Prompts, Middleware & KB** - 提示词、中间件、知识库配置化
- [ ] **Phase 5: App Slim & Integration** - app.py 瘦身 + 端到端验证
